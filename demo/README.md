# Demo

`demo.mp4` is a 2 minute 6 second guided walkthrough generated from repository screenshots and architecture diagrams by `scripts/generate_demo.py`. It covers ingestion, modality-aware extraction, chunking, hybrid retrieval, reranking, cited answers, evaluation, and deployment.

`demo.gif` is a 46-second recruiter-friendly workflow preview generated by `scripts/generate_demo_gif.py`. It highlights upload, querying, retrieval, grounded answers, citations, and retrieved chunks.

The retrieval path uses BGE-small embeddings, persistent FAISS vector search, BM25 candidate fusion, and MS MARCO MiniLM reranking. Local evaluation uses SentenceTransformers; the Render demo uses memory-efficient FastEmbed ONNX equivalents.

For a live demo, start the API and frontend, then open `http://localhost:5173`.

Regenerate the video with:

```bash
python -m pip install -r scripts/requirements.txt
python scripts/generate_demo.py
python scripts/generate_demo_gif.py
```
